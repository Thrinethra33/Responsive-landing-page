# Responsive Landing Page with Interactive Navigation

This repository contains the solution for **Task 01** of the SkillCraft Technology internship. The project focuses on creating a visually appealing, responsive landing page featuring an interactive and dynamic navigation menu.

## 🚀 Project Features

* **Responsive Design:** Optimized for seamless viewing across various devices, including desktops, tablets, and smartphones.
* **Fixed Navigation Bar:** The menu is pinned to the top of the viewport (`position: fixed`), ensuring it remains accessible while scrolling.
* **Scroll-Activated Styling:** Utilizing JavaScript to listen to scroll events, the navbar background color or layout dynamically transforms once the user scrolls past a designated threshold.
* **Interactive Hover Effects:** Smooth CSS transitions apply styled states (color shifts, underlines, etc.) when users hover over individual menu items.

## 🛠️ Technologies Used

* **HTML5:** Structured the layout and semantic elements of the landing page.
* **CSS3:** Flexbox/Grid for responsiveness, standard layout positioning, transitions, and hover state styling.
* **JavaScript (ES6):** Used to detect scroll positioning and toggle classes dynamically on the navigation element.
